NB_COMPAS.py: This file contains the na•ve bayes classifier implementation that reads the Compas dataset and prints the classification report on the predictions.2. KNN_COMPAS.py: This file contains the KNN classifier implementation that reads the Compas dataset and prints the classification report on the predictions. K = 5 was used for implementation. MLE_COMPAS.py: This file contains the maximum likelihood estimate classifier implementation that reads the Compas dataset and prints the classification report on the predictions.These files should be present outside the compas_dataset directory to run.The directory structure to be followed:yp2607_HW1   |   |---compas_dataset   |	|--- propublicaTrain.csv   |	|--- propublicaTest.csv   |   |--- NB_COMPAS.py   |--- KNN_COMPAS.py   |--- MLE_COMPAS.py-------------------------------------------------------------------------------------------------------------------------------Classifiers for Handwritten Digits recognition1.
